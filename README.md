# CarSphere

CarSphere is a premium car dealership web application offering a curated collection of luxury vehicles, flexible financing options, and expert support. Explore, compare, and find your perfect car with ease.

## Features

- **Luxury Car Catalog:** Browse top brands like BMW, Audi, Porsche, Lamborghini, Toyota, and Honda.
- **Advanced Search & Filters:** Find cars by brand, model, and price range.
- **Car Comparison:** Select up to 3 cars and compare their features side-by-side.
- **Financing & Warranty:** Discover flexible financing plans and warranty schemes.
- **Showrooms:** Locate CarSphere showrooms and schedule visits.
- **Contact & Support:** Reach out for inquiries or support via the contact form.
- **EMI Calculator:** Estimate monthly payments for car loans.
- **Financing Partners:** View our trusted banking partners for car financing.

## Technologies Used

- HTML5, CSS3, Bootstrap 5
- JavaScript (with jQuery)
- Responsive design for all devices

## Getting Started

1. Clone or download the repository.
2. Open `index.html` in your browser.
3. All assets are included locally; no server setup required.

## Folder Structure

- `assets/css/` - Stylesheets
- `assets/js/` - JavaScript files
- `assets/images/` - Car images, logos, showpieces, financing partners
- `assets/blender/` - 3D assets (optional)
